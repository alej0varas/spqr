ScrollArea class holds details for a ScrollArea Contains a graphical area that has a scroll bar on it's RHS Call with gui pointer, x/y positon, width and height and image *NOTE*: in the x size, you must allow for the fact that this widget will add a scrollbar on the rhs of a given size! height of widget *MUST BE* >61 pixels or funny gfx effects will occur border area, if true, adds 1 pixel on the lhs and 2 on the y axis
