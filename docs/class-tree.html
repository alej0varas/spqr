    • scripts.spqr_widgets.CSeperator: Seperator class states and stores details for a seperator
    • scripts.spqr_widgets.CSlider: Slider class states and stores details for a slider Call with following parameters: gui - normal gui pointer, x/y pos of widget in window width - width of entire widget, start - value on lhs of widget end - value on rhs of widget, initial - initial value
  • scripts.spqr_window.CWindow: Base window class Call with gui - pointer the gui, x,y - position width,height, title - text on rop of window, draw - wether to place on screen or not
