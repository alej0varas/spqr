         CItemList

Call the init routine with the following parameters: lgui - a pointer to the gfx sub-system: x,y the position then a list of lists, defined in the following way: The first list lets the routine know wether each element of the rest of the lists is text (True) or an image (False) The second list gives the text headers for the columns Of the rest of the lists, each seperate one contains all the elements needed to draw 1 column of the ScrollArea list Then comes an array of routines to sort the data Next comes a list of id numbers, used when sorting the data Finally, the last param tells you what height you would like the element to take up (including the header height)
